Uses https://github.com/SawyerHood/dev-browser to browse web pages.

It only takes one argument: the URL to visit.

## Workflow

1. Install `dev-browser` per the README here: https://github.com/SawyerHood/dev-browser/blob/main/README.md
2. Start `dev-browser`: `dev-browser --connect`
3. Use `dev-browser --help` to output `dev-browser` options and usage
   to launch a browser and do things.
